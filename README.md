### Setup


Install the dependencies and devdependencies.

```sh
  cd social-app

  cd frontend

  npm install

  cd ..

  cd backend

  npm install

```

To start react server

```sh
cd frontend

npm run dev
```

To start node server

```sh
cd backend

npm run dev

```

Add this to your .env file in root of backend folder.

```sh

ACCOUNTSID=
AUTHTOKEN=
VERIFYSID=
ORIGIN=
FACEBOOK_CLIENT_ID=
FACEBOOK_CLIENT_SECRET=
FACEBOOK_CALLBACK_URL=


```
